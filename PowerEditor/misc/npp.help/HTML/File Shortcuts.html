Shortcuts for the File menu

Default shortcuts

Below is the list of predefined keyboard and mouse shortcuts for Notepad++. They can be freely configured using the shortcut mapper, as well as all the commands which were not bound to shortcuts by default. This list does not include standard shortcuts provided by Notepad++-related plugins.



File menu
Shortcut Action
Ctrl-O Open File
Ctrl-N New File
Ctrl-S Save File
Ctrl-Alt-S Save As
Ctrl-Shift-S Save All
Ctrl-P Print
Alt-F4 Exit
Ctrl-Tab Next Document (also shows list of open files). Can be disabled - see Settings/Preferences/Global.
Ctrl-Shift-Tab Previous Document (also shows list of open files). Can be disabled - see above.
Ctrl-W Close Current Document
